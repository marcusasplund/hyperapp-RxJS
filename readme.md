## installation

````bash
    $ git clone https://github.com/marcusasplund/hyperapp-RxJS.git

    $ cd hyperapp-RxJS

    $ yarn

    $ yarn start
````

Open up application at http://localhost:4000/ in browser

## build a release

````bash
    $ yarn run build
````
This will generate a release directory with your minified/rev'd assets.
